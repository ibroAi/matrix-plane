# matrix-plane

Monorepo voor ibro-cli en ai-kit.

## Packages

| Package | Beschrijving |
|---------|-------------|
| `@matrix-plane/ibro-cli` | CLI tool — installeert ai-kit modules in projecten |
| `@matrix-plane/ai-kit` | Content registry — governance folder templates |

## Setup

```bash
npm install
npm run build
```
