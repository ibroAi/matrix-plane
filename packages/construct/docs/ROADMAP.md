# ai-kit — Roadmap

## Vision

ai-kit exists to provide **reliable, reusable structure** for AI-assisted projects.

It is not a framework.
It is not an agent system.

It is a library of *thinking constraints*.

---

## v0.1 — Foundation

Goal:
- establish stable module structure
- define governance layout
- prove drop-in model

Includes:
- `ai` module
- `ai-code` module
- registry format
- documentation baseline

Non-goals:
- automation
- plugin system
- version pinning
- dynamic behavior

---

## v0.2 — Maturity

- richer module metadata
- better documentation
- optional presets
- improved consistency

---

## Future

- community modules
- validation rules
- compatibility checks
- optional tooling support

---

Rule:
> If a module reduces clarity, it does not belong here.
