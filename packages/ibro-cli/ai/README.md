# /ai

Note: This repo uses the same governance structure shipped in `ibroAi/ai-kit` modules.

This folder defines how AI is allowed to behave in this project.
It is intentionally explicit and opinionated.

- governance: hard rules the AI must follow
- templates: consistent prompt/task/report templates
- work: predictable work area for tasks, validations, reports
- code: code-repo add-ons (snapshots, review rules)

This is not an agent. It’s a contract to make AI output reproducible and auditable.
